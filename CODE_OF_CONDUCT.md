# Code of Conduct

## Our pledge
We want iPlus to be a friendly, welcoming place to learn and build — for everyone,
regardless of experience level, background, or identity.

## Expected behavior
- Be kind and assume good intent.
- Give and accept constructive feedback gracefully.
- Respect differing viewpoints; critique ideas, not people.
- Help newcomers — everyone's first PR is a big deal.

## Unacceptable behavior
- Harassment, insults, or discriminatory remarks.
- Personal or political attacks.
- Publishing others' private information without consent.
- Any conduct that would reasonably make someone unwelcome.

## Scope
This applies in all project spaces — issues, pull requests, discussions — and when
representing the project elsewhere.

## Enforcement
If you experience or witness unacceptable behavior, please open a confidential report
by emailing the maintainers (or, if you're comfortable, open an issue). Maintainers may
edit, remove, or reject contributions and comments that violate this code, and may
temporarily or permanently ban any contributor for behavior they deem inappropriate.

This Code of Conduct is adapted from the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org).
